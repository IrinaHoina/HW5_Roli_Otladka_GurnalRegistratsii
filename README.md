# Выполнение заданий по теме "Пользователи и отладка"

## Задача 1. Создание ролей и пользователей

### Что сделано:

1. **Создана роль «ПолныеПрава»**
   - Включены все права на все объекты метаданных
   - Права назначены через классы («Справочники», «Документы», «Отчеты», «Обработки» и т.д.)
   - Установлено предписание установки прав для новых объектов
   - Роль назначена основной для конфигурации

2. **Создана роль «БазовыеПрава»**
   - Права на просмотр и чтение всех данных конфигурации
   - Право на запуск тонкого клиента
   - Право на запуск веб-клиента
   - Без права на редактирование

3. **Создана роль «ДобавлениеИзменениеКонтрагентов»**
   - Право на интерактивное добавление и редактирование справочника «Контрагенты»
   - Других прав (на запуск клиента, просмотр других объектов) нет

4. **Создана роль «ДобавлениеИзменениеСотрудников»**
   - Право на интерактивное добавление и редактирование справочника «Сотрудники»
   - Других прав нет

5. **Созданы пользователи информационной базы:**

| Пользователь | Роли | Что может делать |
|:---|:---|:---|
| Администратор | ПолныеПрава | Редактировать любые данные |
| БазовыйПользователь | БазовыеПрава | Только просматривать данные |
| РедакторКонтрагентов | БазовыеПрава + ДобавлениеИзменениеКонтрагентов | Редактировать контрагентов, просматривать остальное |
| РедакторСотрудников | БазовыеПрава + ДобавлениеИзменениеСотрудников | Редактировать сотрудников, просматривать остальное |

6. **Выполнено тестирование:**
   - Конфигурация запущена в режиме «Предприятия» под каждым пользователем
   - Права работают корректно: администратор редактирует всё, базовый пользователь только просматривает, остальные редактируют только свои объекты

7. **Информационная база выгружена в файл `.dt`**
